Congratulations! That's all! You have just learned all the very basics of SuperMemo!

Now you will need to trust the program for some time before you can see the effects of your work. Your trust will grow if you know how SuperMemo works (see: Principles of SuperMemo).

Use the Learn button (Ctrl+L) daily to make sure you do not forget what you have learned.
Use Add new (Alt+A) to add new questions and answers.

Do not forget to back-up your learning material from time to time. Use File : Copy collection (Ctrl+Shift+C) from the main menu to copy your material to some location on your hard disk. Or better yet: on a DVD-R or a flash drive.
If you cannot see this option, you need to switch SuperMemo to a higher level that exposes some extra program functionality. E.g. to switch SuperMemo from the Beginner level (you are most likely in) to the next Basic level (in which the Copy collection menu item becomes visible), select File : Level : Beginner.
Remember that you have two B-choices: Back up or Be sorry!
Please take a minute to read: How to back up your learning material in SuperMemo.
